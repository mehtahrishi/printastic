# **App Name**: Printastic

## Core Features:

- Product Catalog: Display a catalog of available prints with details, pricing, and images.
- Shopping Cart: Enable users to add prints to a shopping cart and manage their selections.
- Checkout Process: Implement a secure checkout process for users to enter shipping information and payment details.
- Order Management (Admin): Provide an admin interface to view, manage, and fulfill customer orders.
- Product Management (Admin): Allow admins to add, edit, and remove prints from the product catalog.
- Customer Management (Admin): Allow admins to manage customer data, view order history, and track customer interactions.
- Sales Dashboard (Admin): Provide a dashboard with sales revenue, insights, and analytics to track performance.
- Dummy Data Generation: Automatically generate a dataset of dummy products to populate the storefront.
- Print Preview: Allow users to preview how their selected print will look on different surfaces or in different settings.
- Wishlist: Allow users to save products to a wishlist for later purchase.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) for trust and professionalism.
- Background color: Light gray (#F0F4F7) for a clean and modern feel.
- Accent color: Muted orange (#D4A373) for highlighting calls to action.
- Body and headline font: 'Inter', a sans-serif font with a modern, machined look; suitable for both headlines and body text
- Use minimalist line icons for navigation and product categories.
- Implement a grid-based layout to showcase prints in an organized and visually appealing manner.
- Incorporate subtle transition animations on hover and page load for a polished user experience.
